Java Script

JavaScript คือ ภาษาโปรแกรมมิ่งที่ใช้ในการพัฒนาเว็บไซต์ เพื่อเพิ่มความสามารถและฟังก์ชันต่าง ๆ ให้กับหน้าเว็บ เช่น การตอบสนองต่อการคลิกของผู้ใช้ การตรวจสอบข้อมูลในฟอร์ม หรือการแสดงผลข้อมูลแบบไดนามิก โดย JavaScript สามารถทำงานร่วมกับ HTML และ CSS ได้อย่างดีเยี่ยม ทำให้เว็บไซต์มีความน่าสนใจและใช้งานง่ายขึ้น

รูปแบบการเขียน JavaScript

การเขียน JavaScript มีอยู่ 2 ประเภทหลัก ๆ คือ

  • Internal JavaScript: การเขียนโค้ด JavaScript ภายในแท็ก <script> ในไฟล์ HTML โดยตรง
  • External JavaScript: การเขียนโค้ด JavaScript ในไฟล์แยกต่างหากที่มีนามสกุล .js และเชื่อมโยงกับไฟล์ HTML ผ่านแท็ก <script src="ชื่อไฟล์.js"></script>

การแสดงผล JavaScript

JavaScript สามารถแสดงผลได้หลายวิธี เช่น

  • alert(): ใช้แสดงข้อความในหน้าต่างป๊อปอัพ เช่น alert("Hello, World!");
  • document.write(): ใช้เขียนข้อความลงในหน้าเว็บ เช่น document.write("Welcome to JavaScript!");
  • console.log(): ใช้แสดงข้อความในคอนโซลของเบราว์เซอร์ เช่น console.log("This is a log message.");
    • console.error(): ใช้แสดงข้อความแสดงข้อผิดพลาดในคอนโซล
    • console.warn(): ใช้แสดงข้อความเตือนในคอนโซล

การเขียนคอมเมนท์ใน JavaScript

การเขียนคอมเมนท์ใน JavaScript มีสองรูปแบบหลัก ๆ คือ

  • คอมเมนท์แบบบรรทัดเดียว: ใช้เครื่องหมาย // เพื่อเขียนคอมเมนท์ในบรรทัดเดียว
  • คอมเมนท์แบบหลายบรรทัด: ใช้เครื่องหมาย /* และ */ เพื่อเขียนคอมเมนท์ที่ครอบคลุมหลายบรรทัด

การสร้างตัวแปรใน JavaScript

ใน JavaScript เราสามารถสร้างตัวแปรได้โดยใช้คำสั่ง

  • var: ใช้สำหรับสร้างตัวแปรที่สามารถเปลี่ยนค่าได้และมีขอบเขตแบบฟังก์ชัน
  • let: ใช้สำหรับสร้างตัวแปรที่สามารถเปลี่ยนค่าได้และมีขอบเขตแบบบล็อก
  • const: ใช้สำหรับสร้างตัวแปรที่ไม่สามารถเปลี่ยนค่าได้หลังจากที่ถูกกำหนดค่าแล้ว และมีขอบเขตแบบบล็อก

ชนิดของข้อมูลใน JavaScript

JavaScript มีชนิดของข้อมูลหลัก ๆ ดังนี้

  • Number: ใช้สำหรับเก็บค่าตัวเลข เช่น 42, 3.14
  • String: ใช้สำหรับเก็บข้อความ เช่น "Hello, World!"
  • Boolean: ใช้สำหรับเก็บค่าความจริง เช่น true หรือ false
  • Object: ใช้สำหรับเก็บข้อมูลที่มีโครงสร้างซับซ้อน เช่น {name: "John", age: 30}
  • Array: ใช้สำหรับเก็บชุดของข้อมูล เช่น [1, 2, 3, 4, 5]
  • null: ใช้เพื่อแสดงว่าตัวแปรไม่มีค่า
  • undefined: ใช้เพื่อแสดงว่าตัวแปรยังไม่ได้รับการกำหนดค่า
  • typeof: ใช้เพื่อแสดงประเภทของข้อมูลของตัวแปร

การแปลงชนิดข้อมูล

JavaScript มีฟังก์ชันสำหรับการแปลงชนิดข้อมูล เช่น

  • String(): แปลงค่าต่าง ๆ เป็นสตริง
  • Number(): แปลงค่าต่าง ๆ เป็นตัวเลข
  • Boolean(): แปลงค่าต่าง ๆ เป็นบูลีน

ตัวดำเนินการทางคณิตศาสตร์

JavaScript มีตัวดำเนินการทางคณิตศาสตร์หลัก ๆ ดังนี้

  • บวก (+): ใช้สำหรับการบวกตัวเลข
  • ลบ (-): ใช้สำหรับการลบตัวเลข
  • คูณ (*): ใช้สำหรับการคูณตัวเลข
  • หาร (/): ใช้สำหรับการหารตัวเลข
  • โมดูลัส (%): ใช้สำหรับหาค่าเศษจากการหาร
  • เพิ่มค่า (+=): ใช้สำหรับเพิ่มค่าตัวแปร
  • ลดค่า (-=): ใช้สำหรับลดค่าตัวแปร

ตัวดำเนินการเปรียบเทียบ

JavaScript มีตัวดำเนินการเปรียบเทียบหลัก ๆ ดังนี้

  • เท่ากับ (==): ใช้สำหรับตรวจสอบความเท่ากันของค่าตัวแปร
  • ไม่เท่ากับ (!=): ใช้สำหรับตรวจสอบความไม่เท่ากันของค่าตัวแปร
  • มากกว่า (>): ใช้สำหรับตรวจสอบว่าค่าตัวแปรหนึ่งมากกว่าค่าอีกตัวแปรหนึ่ง
  • น้อยกว่า (<): ใช้สำหรับตรวจสอบว่าค่าตัวแปรหนึ่งน้อยกว่าค่าอีกตัวแปรหนึ่ง
  • มากกว่าหรือเท่ากับ (>=): ใช้สำหรับตรวจสอบว่าค่าตัวแปรหนึ่งมากกว่าหรือเท่ากับค่าอีกตัวแปรหนึ่ง
  • น้อยกว่าหรือเท่ากับ (<=): ใช้สำหรับตรวจสอบว่าค่าตัวแปรหนึ่งน้อยกว่าหรือเท่ากับค่าอีกตัวแปรหนึ่ง
  • เท่ากับแบบเข้มงวด (===): ใช้สำหรับตรวจสอบความเท่ากันของค่าตัวแปรและชนิดข้อมูล
  • ไม่เท่ากับแบบเข้มงวด (!==): ใช้สำหรับตรวจสอบความไม่เท่ากันของค่าตัวแปรหรือชนิดข้อมูล

ตัวดำเนินการตรรกศาสตร์

JavaScript มีตัวดำเนินการตรรกศาสตร์หลัก ๆ ดังนี้

  • AND (&&): ใช้สำหรับตรวจสอบว่าทั้งสองเงื่อนไขเป็นจริง
  • OR (||): ใช้สำหรับตรวจสอบว่าอย่างน้อยหนึ่งเงื่อนไขเป็นจริง
  • NOT (!): ใช้สำหรับกลับค่าความจริงของเงื่อนไข

ตัวดำเนินการเพิ่มค่า

JavaScript มีตัวดำเนินการเพิ่มค่าหลัก ๆ ดังนี้

  • เพิ่มค่า ( ++ ): ใช้สำหรับเพิ่มค่าตัวแปรทีละ 1
  • ลดค่า ( -- ): ใช้สำหรับลดค่าตัวแปรทีละ 1

โครงสร้างควบคุมแบบมีเงื่อนไข

JavaScript มีโครงสร้างควบคุมแบบมีเงื่อนไขหลัก ๆ ดังนี้

  • if: ใช้สำหรับตรวจสอบเงื่อนไขและดำเนินการถ้าเงื่อนไขเป็นจริง
  • else: ใช้สำหรับดำเนินการเมื่อเงื่อนไขใน if เป็นเท็จ
  • else if: ใช้สำหรับตรวจสอบเงื่อนไขเพิ่มเติมเมื่อเงื่อนไขใน if เป็นเท็จ
  • switch:case ใช้สำหรับเลือกดำเนินการตามค่าของตัวแปร

คำสั่ง LOOP

JavaScript มีคำสั่ง LOOP หลัก ๆ ดังนี้

  • for: ใช้สำหรับวนลูปตามจำนวนครั้งที่กำหนด
  • while: ใช้สำหรับวนลูปตราบใดที่เงื่อนไขเป็นจริง
  • do...while: ใช้สำหรับวนลูปอย่างน้อยหนึ่งครั้งและตรวจสอบเงื่อนไขหลังจากนั้น

break และ continue

ใน JavaScript คำสั่ง break และ continue ใช้ในการควบคุมการทำงานของลูป ดังนี้:

  • break: ใช้เพื่อหยุดการทำงานของลูปทันทีเมื่อคำสั่งนี้ถูกเรียกใช้ และออกจากลูปนั้น
  • continue: ใช้เพื่อข้ามการทำงานในรอบปัจจุบันของลูป และไปยังรอบถัดไปทันที

ค่าของ null, undefined และ NaN

ใน JavaScript มีค่าพิเศษสามค่า ได้แก่ null, undefined และ NaN ซึ่งมีความหมายและการใช้งานที่แตกต่างกันดังนี้:

  • null: เป็นค่าที่แสดงถึงการไม่มีค่า หรือไม่มีวัตถุใด ๆ ถูกกำหนดให้กับตัวแปร มักใช้เพื่อแสดงว่าตัวแปรนั้นว่างเปล่าโดยเจตนา
  • undefined: เป็นค่าที่แสดงว่าตัวแปรยังไม่ได้รับการกำหนดค่า หรือไม่ได้ถูกประกาศเลย เมื่อเราประกาศตัวแปรแต่ไม่กำหนดค่าให้กับมัน ตัวแปรนั้นจะมีค่าเป็น undefined
  • NaN (Not a Number): เป็นค่าที่แสดงถึงผลลัพธ์ที่ไม่ใช่ตัวเลข มักเกิดขึ้นเมื่อมีการดำเนินการทางคณิตศาสตร์ที่ไม่สามารถคำนวณได้ เช่น การหารด้วยศูนย์ หรือการแปลงสตริงที่ไม่ใช่ตัวเลขเป็นตัวเลข

ฟังก์ชั่น

ฟังก์ชั่นใน JavaScript คือ กลุ่มของคำสั่งที่ถูกจัดกลุ่มไว้เพื่อทำงานเฉพาะเจาะจง ฟังก์ชั่นช่วยให้เราสามารถเรียกใช้งานคำสั่งเหล่านั้นได้หลายครั้งโดยไม่ต้องเขียนโค้ดซ้ำ ฟังก์ชั่นสามารถรับค่าพารามิเตอร์และส่งค่ากลับได้

  • การประกาศฟังก์ชั่น: ใช้คำสั่ง function ตามด้วยชื่อฟังก์ชั่นและวงเล็บ เช่น 
    function greet(name) {
    return "Hello, " + name + "!";
}
  • การเรียกใช้งานฟังก์ชั่น: ใช้วงเล็บหลังชื่อฟังก์ชั่นพร้อมกับส่งค่าพารามิเตอร์ เช่น 
    let message = greet("John");
console.log(message); // Output: Hello, John!
