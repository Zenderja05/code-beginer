HTML เบื้องต้น

HTML (HyperText Markup Language) คือ ภาษามาร์กอัปที่ใช้ในการสร้างและจัดโครงสร้างเนื้อหาบนเว็บเพจ HTML ประกอบด้วยแท็กต่างๆ ที่ใช้ในการกำหนดรูปแบบและการแสดงผลของเนื้อหา เช่น ข้อความ รูปภาพ ลิงก์ ตาราง และอื่นๆ HTML เป็นภาษาที่ง่ายต่อการเรียนรู้และใช้งาน ทำให้ผู้เริ่มต้นสามารถสร้างเว็บเพจได้อย่างรวดเร็ว

HTML5

HTML5 เป็นเวอร์ชันล่าสุดของ HTML ที่มีการเพิ่มฟีเจอร์และความสามารถใหม่ๆ เพื่อรองรับการพัฒนาเว็บแอปพลิเคชันที่ทันสมัย HTML5 มีการสนับสนุนการเล่นวิดีโอและเสียงโดยไม่ต้องใช้ปลั๊กอิน การสร้างกราฟิกด้วยแท็ก <canvas> และการจัดการข้อมูลด้วยแท็ก <data> เป็นต้น HTML5 ยังเน้นการใช้งานร่วมกับ CSS3 และ JavaScript เพื่อสร้างประสบการณ์ผู้ใช้ที่ดีขึ้น

meta tack

  • <meta name="keywords" content=""> ใช้ระบุคำสำคัญที่เกี่ยวข้องกับเนื้อหาของเว็บเพจ เพื่อช่วยในการค้นหา
  • <meta name="description" content=""> ใช้ระบุคำอธิบายสั้นๆ เกี่ยวกับเนื้อหาของเว็บเพจ ซึ่งจะแสดงในผลการค้นหาของเครื่องมือค้นหา
  • <meta name="author" content=""> ใช้ระบุชื่อผู้เขียนหรือผู้สร้างเว็บเพจ
  • <meta charset="UTF-8"> ใช้กำหนดการเข้ารหัสตัวอักษรของเว็บเพจเป็น UTF-8 ซึ่งรองรับตัวอักษรหลากหลายภาษา

แท็ก p

แท็ก <p> ใช้สำหรับสร้างย่อหน้าของข้อความใน HTML โดยจะมีการเว้นวรรคก่อนและหลังย่อหน้าเพื่อความชัดเจน

แท็ก a

แท็ก <a> ใช้สำหรับสร้างลิงก์ไปยังหน้าเว็บอื่นหรือส่วนต่างๆ ภายในหน้าเดียวกัน โดยใช้แอตทริบิวต์ href เพื่อระบุ URL ของลิงก์

แท็ก img

แท็ก <img> ใช้สำหรับแทรกรูปภาพในเว็บเพจ โดยใช้แอตทริบิวต์ src เพื่อระบุที่อยู่ของรูปภาพ และ alt เพื่อระบุข้อความสำรองในกรณีที่รูปภาพไม่สามารถโหลดได้

แท็ก div

แท็ก <div> ใช้สำหรับสร้างกลุ่มขององค์ประกอบ HTML เพื่อจัดการรูปแบบและการจัดวางในหน้าเว็บ โดยมักใช้ร่วมกับ CSS เพื่อกำหนดสไตล์

แท็ก span

แท็ก <span> ใช้สำหรับกำหนดส่วนของข้อความหรือองค์ประกอบภายในบรรทัดเดียวกัน โดยมักใช้ร่วมกับ CSS เพื่อกำหนดสไตล์เฉพาะส่วน

แท็กที่ใช้ร่วมกับ p

  • <strong> ใช้เน้นข้อความให้เด่นชัดขึ้น โดยมักจะแสดงเป็นตัวหนา
  • <em> ใช้เน้นข้อความโดยการทำให้ข้อความเอียง เพื่อแสดงความสำคัญหรือเน้นเสียง
  • <br> ใช้สำหรับขึ้นบรรทัดใหม่ภายในย่อหน้า โดยไม่ต้องเริ่มย่อหน้าใหม่
  • <mark> ใช้เน้นข้อความโดยการทำให้ข้อความมีพื้นหลังสีเหลือง เพื่อดึงดูดความสนใจ
  • <del> ใช้แสดงข้อความที่ถูกลบออก โดยมักจะแสดงเป็นข้อความที่มีเส้นขีดผ่านกลาง
  • <ins> ใช้แสดงข้อความที่ถูกเพิ่มเข้ามาใหม่ โดยมักจะแสดงเป็นข้อความที่มีเส้นใต้
  • <sub> ใช้สำหรับแสดงข้อความในรูปแบบตัวห้อย เช่น สูตรทางวิทยาศาสตร์หรือคณิตศาสตร์
  • <sup> ใช้สำหรับแสดงข้อความในรูปแบบตัวบน เช่น เลขชี้กำลังหรือหมายเลขบรรทัด
  • <code> ใช้สำหรับแสดงข้อความที่เป็นโค้ดโปรแกรม โดยมักจะแสดงในรูปแบบตัวอักษรที่แตกต่างจากข้อความปกติ
  • <blockquote> ใช้สำหรับแสดงข้อความที่เป็นการอ้างอิงหรือคำพูดจากแหล่งอื่น โดยมักจะแสดงในรูปแบบที่มีการเยื้องซ้าย
  • <q> ใช้สำหรับแสดงข้อความที่เป็นคำพูดสั้นๆ ภายในย่อหน้า โดยมักจะแสดงในรูปแบบที่มีเครื่องหมายคำพูด
  • <abbr> ใช้สำหรับแสดงคำย่อ โดยมักจะแสดงในรูปแบบที่มีเส้นใต้และมีคำอธิบายเมื่อเลื่อนเมาส์ไปเหนือคำย่อ
  • <cite> ใช้สำหรับแสดงชื่อของงานเขียน งานวิจัย หรือแหล่งข้อมูล โดยมักจะแสดงในรูปแบบที่มีการเอียง
  • <dfn> ใช้สำหรับกำหนดคำหรือวลีที่มีความหมายเฉพาะเจาะจงในบริบทของข้อความ โดยมักจะแสดงในรูปแบบที่มีการเน้น
  • <time> ใช้สำหรับแสดงวันที่และเวลาที่เกี่ยวข้องกับข้อความ โดยสามารถระบุรูปแบบวันที่และเวลาได้
  • <var> ใช้สำหรับแสดงตัวแปรในข้อความที่เกี่ยวข้องกับคณิตศาสตร์หรือวิทยาศาสตร์ โดยมักจะแสดงในรูปแบบที่มีการเอียง
  • <s> ใช้สำหรับแสดงข้อความที่ไม่ถูกต้องหรือไม่เป็นที่ยอมรับ โดยมักจะแสดงเป็นข้อความที่มีเส้นขีดผ่านกลาง
  • <u> ใช้สำหรับเน้นข้อความโดยการทำให้ข้อความมีเส้นใต้ เพื่อแสดงความสำคัญหรือเน้นเสียง
  • <i> ทำให้ข้อความเป็นตัวเอียง

การแสดงรายการ(list)

การแสดงรายการใน HTML มีอยู่ 4 ประเภท คือ

  • รายการที่ไม่มีลำดับ (Unordered List)
  • รายการที่มีลำดับ (Ordered List)
  • รายการคำอธิบาย (Description List)
  • การซ้อนรายการ (Nested List)

รายการที่ไม่มีลำดับ (Unordered List)

ใช้แท็ก <ul> สำหรับสร้างรายการที่ไม่มีลำดับ โดยใช้แท็ก <li> สำหรับแต่ละรายการ เช่น

  <li>รายการที่ 2</li>
  <li>รายการที่ 3</li>
</ul>

  • รายการที่ 3

การสร้างตาราง (Table)

การสร้างตารางใน HTML ใช้แท็ก <table> โดยมีแท็กย่อยต่างๆ เพื่อกำหนดส่วนต่างๆ ของตาราง เช่น

  • <tr> ใช้สำหรับสร้างแถวของตาราง (table row)
  • <th> ใช้สำหรับสร้างเซลล์หัวข้อของตาราง (table header)
  • <td> ใช้สำหรับสร้างเซลล์ข้อมูลของตาราง (table data)
  • <caption> ใช้สำหรับสร้างคำอธิบายของตาราง (table caption)
  • <thead> ใช้สำหรับสร้างส่วนหัวของตาราง (table head)
  • <tbody> ใช้สำหรับสร้างส่วนเนื้อหาของตาราง (table body)
  • <tfoot> ใช้สำหรับสร้างส่วนท้ายของตาราง (table footer)

ตัวอย่างการสร้างตาราง

<table border="1">
  <caption>ตารางตัวอย่าง</caption>
  <thead>
    <tr>
      <th>หัวข้อที่ 1</th>
      <th>หัวข้อที่ 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>ข้อมูลที่ 1</td>
      <td>ข้อมูลที่ 2</td>
    </tr>
  </tbody>
</table>

  • cellspacing กำหนดระยะห่างระหว่างเซลล์ในตาราง

การสร้างฟรอม (form)

การสร้างฟอร์มใน HTML ใช้แท็ก <form> โดยมีแอตทริบิวต์ต่างๆ เพื่อกำหนดลักษณะของฟอร์ม เช่น

  • action: กำหนด URL ที่จะส่งข้อมูลฟอร์มไปเมื่อผู้ใช้กดปุ่มส่ง (submit)
  • method: กำหนดวิธีการส่งข้อมูลฟอร์ม เช่น GET หรือ POST
  • enctype: กำหนดรูปแบบการเข้ารหัสข้อมูลฟอร์ม เช่น application/x-www-form-urlencoded หรือ multipart/form-data

ภายในแท็ก <form> สามารถใช้แท็กต่างๆ เพื่อสร้างฟิลด์สำหรับกรอกข้อมูล เช่น

  • <input>: ใช้สำหรับสร้างฟิลด์ต่างๆ เช่น
    1. text ใส่ข้อความ
    2. password ใส่รหัสผ่าน
    3. email ใส่อีเมล
    4. number ใส่ตัวเลข
    5. date ใส่วันที่
    6. tel ใส่เบอร์โทร
    7. color ใส่สี
    8. checkbox เลือกได้หลายตัว
    9. radio เลือกได้ตัวเดียว
    10. file อัพโหลดไฟล์
    11. submit ปุ่มส่งข้อมูล
    12. reset ปุ่มรีเซ็ตข้อมูล
    13. hidden ฟิลด์ที่ซ่อนอยู่
    14. range เลือกค่าจากสไลด์
    15. search ช่องค้นหา
    16. url ใส่ลิงก์
    17. image ปุ่มส่งข้อมูลแบบรูปภาพ
    18. datetime-local ใส่วันที่และเวลา
    19. month ใส่เดือนและปี
    20. time ใส่เวลา
    21. week ใส่สัปดาห์และปี
  • <textarea>: ใช้สำหรับสร้างฟิลด์ข้อความหลายบรรทัด
  • <select>: ใช้สำหรับสร้างเมนูแบบเลื่อนลง (dropdown menu)
  • <button>: ใช้สำหรับสร้างปุ่มที่สามารถกดได้

attriburte form

  • action: กำหนด URL ที่จะส่งข้อมูลฟอร์มไปเมื่อผู้ใช้กดปุ่มส่ง (submit)
  • method: กำหนดวิธีการส่งข้อมูลฟอร์ม เช่น GET หรือ POST
  • enctype: กำหนดรูปแบบการเข้ารหัสข้อมูลฟอร์ม เช่น application/x-www-form-urlencoded หรือ multipart/form-data
  • name: กำหนดชื่อของฟอร์ม
  • target: กำหนดที่ที่จะแสดงผลลัพธ์หลังจากส่งฟอร์ม เช่น _blank, _self, _parent, _top
  • autocomplete: กำหนดว่าฟอร์มควรเปิดใช้งานการเติมข้อความอัตโนมัติหรือไม่ (on หรือ off)
  • novalidate: ใช้เพื่อปิดการตรวจสอบความถูกต้องของฟอร์มเมื่อส่งข้อมูล
  • placeholder ใช้เขียนอธิบายในช่องอินพุด
  • required ใช้บังคับให้กรอกข้อมูลในช่องอินพุด
  • readonly ใช้สำหรับทำให้ช่องอินพุดไม่สามารถแก้ไขได้
  • disabled ใช้สำหรับทำให้ช่องอินพุดไม่สามารถใช้งานได้

ตัวอย่างการสร้างฟอร์ม






















block,inline

Block-level elements (บล็อกเลเวล อิลิเมนต์) คือ อิลิเมนต์ที่เริ่มต้นในบรรทัดใหม่และครอบคลุมความกว้างทั้งหมดของพาเรนต์คอนเทนเนอร์ ตัวอย่างของบล็อกเลเวล อิลิเมนต์ ได้แก่

  • <div>
  • <h1> ถึง <h6>
  • <p>
  • <ul> และ <ol>
  • <li>
  • <table>

Inline elements (อินไลน์ อิลิเมนต์) คือ อิลิเมนต์ที่ไม่เริ่มต้นในบรรทัดใหม่และครอบคลุมเฉพาะเนื้อหาภายในตัวเอง ตัวอย่างของอินไลน์ อิลิเมนต์ ได้แก่

  • <a>
  • <span>
  • <img>
  • <strong>
  • <em>
  • <code>

id,class

id และ class เป็นแอตทริบิวต์ที่ใช้ในการระบุและจัดกลุ่มองค์ประกอบ HTML เพื่อให้สามารถนำไปใช้กับ CSS และ JavaScript ได้อย่างมีประสิทธิภาพ

  • id: ใช้ระบุองค์ประกอบ HTML ที่เฉพาะเจาะจง โดย id ควรมีค่าเป็นเอกลักษณ์ (unique) ภายในหน้าเว็บเดียวกัน สามารถใช้ id เพื่อกำหนดสไตล์หรือเข้าถึงองค์ประกอบผ่าน JavaScript ได้ เช่น <div id="header">... (#)
  • class: ใช้จัดกลุ่มองค์ประกอบ HTML ที่มีลักษณะหรือบทบาทคล้ายกัน โดย class สามารถมีค่าเดียวกันได้ในหลายองค์ประกอบภายในหน้าเว็บเดียวกัน สามารถใช้ class เพื่อกำหนดสไตล์หรือเข้าถึงกลุ่มองค์ประกอบผ่าน JavaScript ได้ เช่น <div class="container">...(.)

semantic tag

Semantic tags (แท็กเชิงความหมาย) คือ แท็ก HTML ที่มีความหมายเฉพาะเจาะจงเกี่ยวกับเนื้อหาที่อยู่ภายในแท็กนั้นๆ ซึ่งช่วยให้โครงสร้างของหน้าเว็บมีความชัดเจนและง่ายต่อการเข้าใจ ทั้งสำหรับผู้ใช้และเครื่องมือค้นหา ตัวอย่างของ semantic tags ได้แก่

  • <header>: ใช้สำหรับส่วนหัวของหน้าเว็บหรือส่วนของเนื้อหา
  • <nav>: ใช้สำหรับเมนูนำทางหรือส่วนที่มีลิงก์ไปยังหน้าต่างๆ
  • <main>: ใช้สำหรับเนื้อหาหลักของหน้าเว็บ
  • <article>: ใช้สำหรับเนื้อหาที่เป็นอิสระและสามารถเผยแพร่ได้ เช่น บทความ ข่าว หรือโพสต์ในบล็อก
  • <section>: ใช้สำหรับแบ่งเนื้อหาออกเป็นส่วนๆ ที่มีความสัมพันธ์กัน
  • <aside>: ใช้สำหรับเนื้อหาที่เกี่ยวข้องแต่ไม่ใช่ส่วนหลัก เช่น แถบด้านข้าง หรือข้อมูลเพิ่มเติม
  • <footer>: ใช้สำหรับส่วนท้ายของหน้าเว็บหรือส่วนของเนื้อหา
  • <figure>: ใช้สำหรับกลุ่มของสื่อ เช่น รูปภาพ แผนภูมิ หรือวิดีโอ พร้อมคำอธิบาย
  • <figcaption>: ใช้สำหรับคำอธิบายของแท็ก <figure>
  • <mark>: ใช้สำหรับเน้นข้อความที่สำคัญ
  • <time>: ใช้สำหรับแสดงวันที่และเวลา

HTML Entity

HTML Entity (เอนทิตี HTML) คือ รหัสพิเศษที่ใช้แทนตัวอักษรหรือสัญลักษณ์ที่มีความหมายพิเศษใน HTML เพื่อป้องกันไม่ให้เบราว์เซอร์ตีความผิด ตัวอย่างเช่น การใช้ & แทนเครื่องหมาย < หรือ > แทนเครื่องหมาย > เป็นต้น

ตัวอย่าง HTML Entity ที่ใช้บ่อย:

  • &nbsp; : ช่องว่าง (non-breaking space)
  • &lt; : เครื่องหมาย < (less than)
  • &gt; : เครื่องหมาย > (greater than)
  • &amp; : เครื่องหมาย & (ampersand)
  • &quot; : เครื่องหมาย " (double quote)
  • &apos; : เครื่องหมาย ' (single quote)
  • &copy; : สัญลักษณ์ลิขสิทธิ์ (©)
  • &reg; : สัญลักษณ์เครื่องหมายการค้า (®)
  • &euro; : สัญลักษณ์ยูโร (€)
  • &pound; : สัญลักษณ์ปอนด์ (£)
  • &yen; : สัญลักษณ์เยน (¥)
  • &cent; : สัญลักษณ์เซ็นต์ (¢)
  • &sect; : สัญลักษณ์มาตรา (§)
  • &para; : สัญลักษณ์ย่อหน้าหรือพารากราฟ (¶)
  • &bull; : สัญลักษณ์หัวข้อย่อย (•)
  • &hellip; : สัญลักษณ์จุดไข่ปลา (…)
  • &mdash; : สัญลักษณ์ขีดยาว (—)
  • &ndash; : สัญลักษณ์ขีดกลาง (–)
  • &trade; : สัญลักษณ์เครื่องหมายการค้า (™)

การเล่นวิดีโอ

การเล่นวิดีโอใน HTML ใช้แท็ก <video> โดยมีแอตทริบิวต์ต่างๆ เพื่อกำหนดลักษณะของวิดีโอ เช่น

  • src: กำหนดที่อยู่ของไฟล์วิดีโอ
  • controls: แสดงแถบควบคุมวิดีโอ เช่น ปุ่มเล่น หยุด และปรับเสียง
  • autoplay: เล่นวิดีโอโดยอัตโนมัติเมื่อโหลดหน้าเว็บ
  • loop: เล่นวิดีโอซ้ำเมื่อจบ
  • muted: ปิดเสียงวิดีโอเริ่มต้น
  • width: กำหนดความกว้างของวิดีโอ
  • height: กำหนดความสูงของวิดีโอ

ตัวอย่างการเล่นวิดีโอในเครื่อง

<video controls width="150" height="300">
  <source src="Download.mp4" type="video/mp4">
</video>

การเล่นเสียง (audio)

การเล่นเสียงใน HTML ใช้แท็ก <audio> โดยมีแอตทริบิวต์ต่างๆ เพื่อกำหนดลักษณะของเสียง เช่น

  • src: กำหนดที่อยู่ของไฟล์เสียง
  • controls: แสดงแถบควบคุมเสียง เช่น ปุ่มเล่น หยุด และปรับเสียง
  • autoplay: เล่นเสียงโดยอัตโนมัติเมื่อโหลดหน้าเว็บ
  • loop: เล่นเสียงซ้ำเมื่อจบ
  • muted: ปิดเสียงเริ่มต้น
  • volume: กำหนดระดับเสียงเริ่มต้น (ค่าระหว่าง 0.0 ถึง 1.0)
