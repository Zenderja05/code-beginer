คู่มือสอน CSS Attributes ครบทุกคำสั่ง พร้อมตัวอย่าง

1. Text Properties (ข้อความ)

ใช้ตกแต่งข้อความ:

p {
color: crimson;
font-size: 20px;
text-align: center;
text-transform: uppercase;
}
นี่คือตัวอย่างข้อความ

2. Background & Color

ใช้ปรับสีและพื้นหลัง:

div {
background-color: teal;
background-image: linear-gradient(90deg, teal, skyblue);
color: white;
}
พื้นหลังแบบไล่เฉดสี

3. Box Model

8. Shadow & Filter

9. Transition

div {
transition: all 0.4s ease;
}
div:hover {
transform: scale(1.2);
}
วางเมาส์ที่นี่

10. Animation

@keyframes move {
0% {transform: translateX(0);}
50% {transform: translateX(80px);}
100% {transform: translateX(0);}
}

div {
animation: move 2s infinite;
}
กำลังเคลื่อนที่

11. Overflow & Visibility

div {
overflow: auto;
visibility: visible;
}
เนื้อหานี้ยาวเกินกล่อง ลองเลื่อนดูเพื่อเห็นผลว่า overflow ทำงานอย่างไร
