CSS

CSS ย่อมาจาก Cascading Style Sheets

CSS ใช้สำหรับตกแต่งเว็บไซต์

CSS สามารถกำหนดสี, ฟอนต์, ขนาด, ระยะห่าง และอื่นๆ

CSS สามารถทำให้เว็บไซต์ดูสวยงามและน่าสนใจมากขึ้น

CSS สามารถทำให้เว็บไซต์ตอบสนองต่ออุปกรณ์ต่างๆ เช่น มือถือ แท็บเล็ต และคอมพิวเตอร์

CSS สามารถทำให้เว็บไซต์โหลดเร็วขึ้นโดยการลดขนาดไฟล์ HTML

CSS สามารถทำให้เว็บไซต์มีการจัดวางที่ดีขึ้นและง่ายต่อการอ่าน

การเชื่อม HTML กับ CSS

การเชื่อม HTML กับ CSS สามารถทำได้ 3 วิธี

  • Inline CSS คือ การใช้ CSS ในแท็กนั้น
  • Internal CSS คือ การใช้ CSS ในแท็ก head
  • External CSS คือการใช้ CSS แบบแยกไฟล์

ตัวอย่างการใช้ CSS

1. Inline CSS

<p style="color:blue; font-size:20px;">นี่คือตัวอย่างการใช้ Inline CSS</p>

นี่คือตัวอย่างการใช้ Inline CSS

2. Internal CSS

<style> .internal-css { color: green; font-size: 18px; } </style> <p class="internal-css">นี่คือตัวอย่างการใช้ Internal CSS</p>

นี่คือตัวอย่างการใช้ Internal CSS

3. External CSS

<link rel="stylesheet" href="styles.css"> <p class="external-css">นี่คือตัวอย่างการใช้ External CSS</p>

นี่คือตัวอย่างการใช้ External CSS

CSS selector

CSS selector คือ ส่วนที่ใช้เลือกองค์ประกอบ HTML ที่ต้องการนำไปตกแต่งด้วย CSS

CSS selector มีหลายประเภท เช่น element selector, class selector, id selector, attribute selector และ pseudo-class selector

ตัวอย่าง CSS selector
  • Element selector: p { color: red; }
  • Class selector: .class-name { font-size: 20px; }
  • ID selector: #id-name { background-color: yellow; }
  • Attribute selector: a[target="_blank"] { text-decoration: underline; }
  • Pseudo-class selector: a:hover { color: green; }

CSS media query

CSS media query คือ เทคนิคที่ใช้ในการกำหนดสไตล์ CSS ให้เหมาะสมกับขนาดหน้าจอหรืออุปกรณ์ที่ใช้งาน

CSS media query สามารถใช้ในการสร้างเว็บไซต์ที่ตอบสนองต่ออุปกรณ์ต่างๆ (responsive web design)

CSS media query สามารถกำหนดเงื่อนไขต่างๆ เช่น ขนาดหน้าจอ, ความละเอียดหน้าจอ, ทิศทางการแสดงผล และอื่นๆ

ตัวอย่าง CSS media query

universal selector

universal selector คือ ตัวเลือกที่ใช้เลือกทุกองค์ประกอบในเอกสาร HTML

โดยใช้เครื่องหมายดอกจัน (*) เป็นตัวเลือก

มักใช้ในการรีเซ็ตสไตล์เริ่มต้นของเบราว์เซอร์

ตัวอย่าง universal selector

* { margin: 0; padding: 0; box-sizing: border-box; }(กดเอาคอมเมนท์ในตัวโค้ดก่อนเพื่อดูตัวอย่าง)

CSS attriburte

attriburte font

  • font-family: กำหนดฟอนต์ที่ใช้ในข้อความ
  • font-size: กำหนดขนาดของฟอนต์
  • font-weight: กำหนดความหนาของฟอนต์ เช่น normal, bold, bolder, lighter หรือค่าตัวเลข 100-900
  • font-style: กำหนดลักษณะของฟอนต์ เช่น normal, italic หรือ oblique
  • text-align: กำหนดการจัดตำแหน่งข้อความ เช่น left, right, center หรือ justify
  • text-decoration: กำหนดการตกแต่งข้อความ เช่น none, underline, overline หรือ line-through
  • color: กำหนดสีของข้อความ

attriburte color
  • color: กำหนดสีของข้อความ
  • background-color: กำหนดสีพื้นหลังขององค์ประกอบ
  • border-color: กำหนดสีของเส้นขอบ
  • opacity: กำหนดความโปร่งใสขององค์ประกอบ โดยมีค่าอยู่ระหว่าง 0 (โปร่งใสเต็มที่) ถึง 1 (ทึบเต็มที่)
  • box-shadow: กำหนดเงาของกล่อง โดยสามารถกำหนดค่าต่างๆ เช่น ระยะห่าง, ความเบลอ, สี และอื่นๆ

attriburte text-align
  • left: จัดข้อความชิดซ้าย
  • right: จัดข้อความชิดขวา
  • center: จัดข้อความให้อยู่กึ่งกลาง
  • justify: จัดข้อความให้ชิดทั้งสองด้าน โดยการเพิ่มระยะห่างระหว่างคำ

attriburte width-hight
  • width: กำหนดความกว้างขององค์ประกอบ เช่น 100px, 50%, auto
  • height: กำหนดความสูงขององค์ประกอบ เช่น 100px, 50%, auto
  • max-width: กำหนดความกว้างสูงสุดขององค์ประกอบ เช่น 100px, 50%, none
  • max-height: กำหนดความสูงสูงสุดขององค์ประกอบ เช่น 100px, 50%, none
  • min-width: กำหนดความกว้างต่ำสุดขององค์ประกอบ เช่น 100px, 50%, auto
  • min-height: กำหนดความสูงต่ำสุดขององค์ประกอบ เช่น 100px, 50%, auto

attriburte broder
  • border-width: กำหนดความหนาของเส้นขอบ เช่น 1px, 0.5em, thin, medium, thick
  • border-style: กำหนดลักษณะของเส้นขอบ เช่น none, solid, dashed, dotted, double, groove, ridge, inset, outset
  • border-color: กำหนดสีของเส้นขอบ เช่น red, #ff0000, rgb(255,0,0)
  • border-radius: กำหนดความโค้งมนของมุมเส้นขอบ เช่น 5px, 50%, 1em
  • border: กำหนดเส้นขอบโดยรวมความหนา, ลักษณะ และสี เช่น 1px solid red

attriburte box-model
  • margin: กำหนดระยะห่างภายนอกขององค์ประกอบ เช่น 10px, 1em, auto
    • margin-top: กำหนดระยะห่างด้านบน
    • margin-right: กำหนดระยะห่างด้านขวา
    • margin-bottom: กำหนดระยะห่างด้านล่าง
    • margin-left: กำหนดระยะห่างด้านซ้าย
  • padding: กำหนดระยะห่างภายในขององค์ประกอบ เช่น 10px, 1em, 0
    • padding-top: กำหนดระยะห่างด้านบน
    • padding-right: กำหนดระยะห่างด้านขวา
    • padding-bottom: กำหนดระยะห่างด้านล่าง
    • padding-left: กำหนดระยะห่างด้านซ้าย
  • border: กำหนดเส้นขอบขององค์ประกอบ โดยรวมความหนา, ลักษณะ และสี เช่น 1px solid red
  • box-sizing: กำหนดวิธีการคำนวณขนาดขององค์ประกอบ เช่น content-box (ค่าเริ่มต้น) หรือ border-box

attriburte background
  • background-color: กำหนดสีพื้นหลังขององค์ประกอบ เช่น red, #ff0000, rgb(255,0,0)
  • background-image: กำหนดภาพพื้นหลังขององค์ประกอบ เช่น url("image.jpg")
  • background-repeat: กำหนดการทำซ้ำของภาพพื้นหลัง เช่น repeat (ค่าเริ่มต้น), no-repeat, repeat-x, repeat-y
  • background-size: กำหนดขนาดของภาพพื้นหลัง เช่น auto (ค่าเริ่มต้น), cover, contain หรือกำหนดขนาดเป็น px หรือ %
  • background-position: กำหนดตำแหน่งของภาพพื้นหลัง เช่น left top (ค่าเริ่มต้น), center center, right bottom หรือกำหนดเป็น px หรือ %
  • background-attachment: กำหนดการเลื่อนของภาพพื้นหลัง เช่น scroll (ค่าเริ่มต้น) หรือ fixed

attriburte float(การลอยของ element แบบเว็บพนัน))
  • float: กำหนดให้องค์ประกอบลอยซ้ายหรือขวา เช่น left, right หรือ none (ค่าเริ่มต้น)
  • clear: กำหนดการหยุดการลอยขององค์ประกอบ เช่น left, right, both หรือ none (ค่าเริ่มต้น)

attriburte display

attriburte position
  • position: กำหนดวิธีการจัดวางองค์ประกอบ เช่น static (ค่าเริ่มต้น), relative, absolute, fixed, sticky
  • top: กำหนดระยะห่างจากด้านบนขององค์ประกอบเมื่อใช้ position ที่ไม่ใช่ static
  • right: กำหนดระยะห่างจากด้านขวาขององค์ประกอบเมื่อใช้ position ที่ไม่ใช่ static
  • bottom: กำหนดระยะห่างจากด้านล่างขององค์ประกอบเมื่อใช้ position ที่ไม่ใช่ static
  • left: กำหนดระยะห่างจากด้านซ้ายขององค์ประกอบเมื่อใช้ position ที่ไม่ใช่ static
  • z-index: กำหนดลำดับชั้นขององค์ประกอบเมื่อมีการซ้อนทับกัน โดยมีค่าเป็นจำนวนเต็มที่มากกว่าจะอยู่ด้านบน

attriburte visibility

  • visible (ค่าเริ่มต้น)
  • hidden (ซ่อนวัตถุ)
  • collapse (ซ่อนวัตถุและไม่ใช้พื้นที่)

attriburte position

คือ กำหนดวิธีการจัดวางองค์ประกอบ เช่น

  • static จัดวางค่าเริ่มต้น
  • relative ใช้ในการจัดวางและกำหนดตำแหน่งวัตถุโดยการนับจากจุดที่วัตถุนั้นๆอยู่
  • absolute ใช้ในการจัดวางและกำหนดตำแหน่งวัตถุโดยการนับจากจุดที่อยู่ใกล้ที่สุดที่มีการกำหนด position เป็น
    • fiixde (ใช้ในการจัดวางวัตถุให้อยู่ตำแหน่งเดิม)
    • sticky (ให้วัตถุติดขอบเมื่อเลื่อนไปถึง)

attriburte z-index

คือ การกำหนดลำดับชั้นขององค์ประกอบเมื่อมีการซ้อนทับกัน

  • มีค่าเป็นจำนวนเต็มที่มากกว่าจะอยู่ด้านบน
  • ค่าเริ่มต้นคือ auto ซึ่งหมายความว่าองค์ประกอบจะถูกจัดวางตามลำดับในเอกสาร HTML
  • ต้องใช้ร่วมกับ position ที่ไม่ใช่ static (เช่น relative, absolute, fixed, sticky)

attriburte !important

คือ การเพิ่มความสำคัญให้กับกฎ CSS ที่กำหนดไว้ โดยใช้คำว่า !important ต่อท้ายค่าของคุณสมบัติ

เมื่อมีการใช้ !important กับกฎ CSS จะทำให้กฎนั้นมีความสำคัญสูงสุดและจะถูกนำไปใช้แทนที่กฎอื่นๆ ที่อาจขัดแย้งกัน

ควรใช้ !important อย่างระมัดระวังและในกรณีที่จำเป็นเท่านั้น เนื่องจากอาจทำให้การจัดการสไตล์ของเว็บไซต์ซับซ้อนขึ้นและยากต่อการบำรุงรักษา

attriburte overflow

คือ การกำหนดวิธีการจัดการกับเนื้อหาที่เกินขนาดของกล่องที่กำหนดไว้

  • visible (ค่าเริ่มต้น): เนื้อหาที่เกินขนาดของกล่องจะแสดงออกมา
  • hidden: เนื้อหาที่เกินขนาดของกล่องจะถูกซ่อน
  • scroll: เพิ่มแถบเลื่อน (scrollbar) เพื่อให้สามารถเลื่อนดูเนื้อหาที่เกินขนาดได้
    • overflow:scroll
    • overflow-x:scroll (เพิ่มแถบเลื่อนแนวนอน)
    • overflow-y:scroll (เพิ่มแถบเลื่อนแนวตั้ง)
  • auto: เพิ่มแถบเลื่อนเมื่อเนื้อหาที่เกินขนาดของกล่อง

attriburte cursor style

คือ การกำหนดลักษณะของเคอร์เซอร์เมื่อวางเมาส์เหนือองค์ประกอบ

  • pointer: เคอร์เซอร์รูปมือชี้ ใช้สำหรับลิงก์หรือปุ่มที่สามารถคลิกได้
  • text: เคอร์เซอร์รูปตัว I ใช้สำหรับข้อความที่สามารถแก้ไขได้
  • move: เคอร์เซอร์รูปสี่ทิศทาง ใช้สำหรับการลากวัตถุ
  • not-allowed: เคอร์เซอร์รูปวงกลมมีเส้นทแยง ใช้สำหรับวัตถุที่ไม่สามารถใช้งานได้
  • crosshair: เคอร์เซอร์รูปกากบาท ใช้สำหรับการเลือกพื้นที่

attriburte box-shadow
